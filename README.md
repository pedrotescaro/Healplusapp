# Healplusapp

Healplusapp is an Android application designed to assist healthcare professionals in managing patient information efficiently. The app provides features for creating and managing anamneses (medical histories), patient records (fichas), and scheduling appointments (agenda).

## Features

*   **Anamnese Management:** Create, view, and manage patient anamneses.
*   **Patient Records:** Keep track of patient records with an easy-to-use interface.
*   **Agenda:** Schedule and manage appointments.
*   **Accessibility:** Includes a high-contrast theme for improved readability.

## Project Structure

The project follows a feature-based packaging structure, promoting a clean and scalable architecture. The main packages are:

*   **data:** Contains data-related classes, such as repositories and data sources.
*   **features:** Contains the different features of the app, such as anamnese, fichas, and agenda.
*   **settings:** Handles user preferences, such as the high-contrast theme.
*   **ui:** Contains UI-related classes, such as activities and fragments.
